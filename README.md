# Rögar Kapağı

Bu proje, Unity (2D URP) kullanılarak geliştirilmekte olan "Rögar Kapağı" adlı bir oyundur.

## Geliştirme Ortamı
- **Oyun Motoru:** Unity
- **Render Pipeline:** Universal Render Pipeline (URP - 2D)
- **Platform:** PC / Mobil (Hedef platforma göre değişebilir)

## Proje Dizini Çevresindeki Yapı
- `Assets/`: Oyunun ana içeriklerinin (scriptler, sahneler, görseller, sesler vb.) bulunduğu klasör.
- `Packages/`: Projeye dahil edilmiş olan paket parçaları.
- `ProjectSettings/`: Projeye ait ayarların tutulduğu yapılandırma dosyaları.

## Kurulum ve Çalıştırma
1. Bu depoyu bilgisayarınıza klonlayın.
2. Unity Hub'ı açın ve projeyi listeye ekleyin (Add project from disk).
3. Doğru Unity sürümü ile projeyi açın.
4. `Assets/Scenes` klasörü içerisinden ana sahneyi bularak oyunu test edebilirsiniz.

## Katkıda Bulunma
Herhangi bir geliştirme veya hata düzeltmesi yapmak isterseniz, lütfen bir dal (branch) oluşturarak değişikliklerinizi yapın ve ardından bir "Pull Request" gönderin.
